# idea-vault 🎫

매일 뉴스를 조사해서 바이브 코딩용 앱 아이디어를 발굴하고, 내 취향을 학습하는 개인 에이전트.

## 매일 아침 (30초)

Claude Code에서 이 저장소를 열고:

```
/idea
```

분야를 바꾸고 싶으면: `/idea 여행, 마케팅`

결과는 `ideas/오늘날짜.md` 에 저장된다.

## 아이디어를 본 후

각 아이디어의 `평가:` 줄을 직접 고친다:
- ⭐ 마음에 듦 / ❌ 별로 / ✅ 실제로 만들었음

## 일주일에 한 번 (선택)

```
/taste
```

쌓인 평가를 분석해서 taste.md를 업데이트한다. 이후의 /idea 는 학습된 취향을 반영한다.

## 구조

- `CLAUDE.md` — 에이전트의 두뇌 (프로세스와 필터 기준)
- `ideas/` — 기억 (날짜별 아이디어 아카이브, 중복 방지에 사용)
- `taste.md` — 취향 (평가에서 학습)
- `.claude/commands/` — /idea, /taste 명령
