# KIKI의 취향

에이전트는 아이디어를 필터링할 때 이 파일을 반드시 읽는다.
아래 내용은 시작값이며, /taste 명령이 평가 데이터를 바탕으로 계속 다듬는다.

## 좋아하는 것

- 생활 속에서 매일 반복되는 작은 불편을 해결하는 앱
- 열자마자 바로 쓸 수 있는 단순한 앱 (설명이 필요 없는 것)
- 기존 인기 앱을 재치있게 비튼 것

## 싫어하는 것

- 로그인, 회원가입, 결제가 필요한 아이디어
- "플랫폼", "커뮤니티", "매칭" 등 사용자가 많이 모여야만 가치가 생기는 아이디어
- 유행어만 있고 구체적인 불편이 없는 아이디어

## 관찰된 패턴 (자동 업데이트 영역)
