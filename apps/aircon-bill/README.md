## 소개

폭염에 에어컨을 하루 종일 틀면서도 "이번 달 전기세 폭탄 아닐까?"를 고지서 나올 때까지 한 달간 알 방법이 없는 불편을 해결하는 앱입니다.

에어컨 소비전력과 하루 사용 시간만 넣으면 **한국 주택용(저압) 하계 누진제**를 반영한 이번 달 예상 전기요금을 즉시 보여줍니다.

- 슬라이더를 움직이는 순간 요금이 실시간으로 바뀝니다
- "에어컨 없을 때 대비 **+149,260원**" 처럼 에어컨 몫만 따로 보여줍니다
- 누진 구간 진입 여부를 색깔 배지로 경고합니다 (1구간 😌 / 2구간 😐 / 3구간 🔥)
- 입력값은 브라우저에 자동 저장되어 다음에 열어도 유지됩니다
- 로그인 없음, 서버 없음, 파일 하나

## 사용 방법

1. 위 실행 링크를 엽니다 (스마트폰도 OK)
2. **에어컨 소비전력**: 제품 라벨의 "냉방 소비전력(W)"을 입력하거나, 프리셋 버튼(벽걸이 6평 / 스탠드 15평 / 스탠드 20평)을 누릅니다
3. **하루 평균 사용 시간** 슬라이더를 조절합니다 — 인버터형은 실제 켠 시간의 70% 정도로 잡으면 현실적입니다
4. **에어컨 빼고 평소 월 사용량**은 모르면 기본값(300kWh) 그대로 두세요
5. 아래 카드에서 이번 달 예상 요금과 누진 구간을 확인합니다

### 로컬에서 실행

설치가 필요 없습니다. `index.html` 파일을 내려받아 더블클릭하면 끝.

## 계산 기준

- 한국전력 주택용(저압) **하계(7~8월)** 누진 요금표: 300kWh / 450kWh 경계, 1000kWh 초과 슈퍼유저 요금
- 기후환경요금, 연료비조정액, 부가가치세(10%), 전력산업기반기금(3.7%) 포함
- 요금표가 개정되면 `index.html` 상단의 `TIERS` 상수만 고치면 됩니다

⚠️ 어디까지나 **추정치**입니다. 실제 고지액은 검침일, 복지 할인, 요금 개정 등에 따라 달라질 수 있어요.
