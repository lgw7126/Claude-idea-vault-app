## 2. 지금넣을까 (주유 타이밍 판정기)
- 평가: (아직 없음)
- 유형: 새로움

- MVP 기능 (딱 하나): 오피넷 공공 API 하나로 국제유가(두바이유) 최근 추이와 전국/내 지역 주유소 평균가 추이를 겹쳐 보여주고, 반영 시차(약 2~3주)를 근거로 "지금 넣어라 / N일 기다려라" 한 줄 판정
- 난이도: 보통
- 영감이 된 뉴스: [최고가 적용하는데 기름값 왜 매일 오르나…유가 급락 언제 반영? (뉴스1)](https://www.news1.kr/industry/general-industry/6131187), [기름값 24.7% 급등…6월 물가 3.2%↑, 30개월만에 최고 (파이낸셜뉴스)](https://www.fnnews.com/news/202607021038529642), [기름 넣으러 왔다가 '깜짝'…2000원 돌파에 "주말 여행도 못가" (머니투데이)](https://www.mt.co.kr/society/2026/04/19/2026041913213784539)
- 자기평가: 9/10 — 지금 이 순간(종전 직후 유가 하락 반영기)에 가장 시의적절하고, 매주 반복되는 지출 결정에 직접 돈을 아껴준다. 판정 근거(국제가→국내가 시차)가 데이터로 검증 가능해서 "근거 없는 계산기" 함정도 없다. 감점 1은 유가 안정기가 오면 효용이 줄어든다는 점.

## 3. 렌탈의 민낯 (가전 구독 총비용 계산기)
- 평가: (아직 없음)
- 유형: 새로움
- 불편: 가전 구독(렌탈) 계약자의 31.4%가 중도해지 위약금을 모른 채 서명하고, "월 3만원" 광고만 보고 계약했다가 총비용이 수백만원인 걸 뒤늦게 아는 피해가 뉴스가 될 정도로 반복된다. 정부의 총비용 표시 의무 확대는 연말에나 시행 — 그 전까지는 소비자가 스스로 계산하는 수밖에 없다
- 타깃: 정수기·에어컨·냉장고 등 가전 구독 계약을 앞두고 매장/전화 상담 중인 소비자
- MVP 기능 (딱 하나): 월 요금·계약기간·위약금율(계약서 숫자 그대로)을 입력하면 → 총 납부액, 일시불 대비 차액, 그리고 "N개월차에 해지하면 위약금 얼마"를 시점별 그래프로 보여주기
- 난이도: 쉬움
- 영감이 된 뉴스: [월 요금만 강조한 가전 구독, 총비용·위약금은 깜깜이 (우먼컨슈머)](https://www.womancs.co.kr/news/articleView.html?idxno=100466), ["언제든 해지? 위약금 폭탄"…대형 가전 구독 '꼼수 영업' 제동 (경기일보)](https://www.kyeonggi.com/article/20260407580471)
- 자기평가: 8/10 — 계산 근거가 "사용자 본인의 계약 조건 입력"이라 여행 위약금 계산기처럼 틀린 답을 줄 위험이 없고, 외부 API 0개로 하루면 만든다. 감점 2는 한 사람이 자주 쓰는 앱은 아니라는 점(계약 시점에 1회) — 대신 그 1회가 수백만원짜리 결정이다.

### 빌드 프롬프트
> 화면 1개짜리 웹앱을 만들어줘. 입력: 월 렌탈료, 계약기간(개월), 등록비/설치비, 중도해지 위약금율(%), 비교용 일시불 가격(선택). 출력: ① 총 납부액 큰 숫자, ② 일시불 대비 +얼마인지, ③ 1개월차부터 만기까지 "이 시점에 해지하면 남은 돈 + 위약금 = 얼마"를 보여주는 막대 그래프. 외부 API·로그인 없이 순수 계산만, 모바일 화면 기준으로.
