내 취향을 학습해서 taste.md를 업데이트해라.

1. ideas/ 폴더의 모든 파일을 읽고, 내가 평가를 표시한 아이디어를 수집해라:
   - ⭐ = 마음에 들었던 아이디어
   - ❌ = 별로였던 아이디어
   - ✅ = 실제로 만든 아이디어 (가장 강한 긍정 신호)
2. 표시된 아이디어들에서 패턴을 찾아라. 유형, 분야, 난이도, 문제의 성격 등.
3. 발견한 패턴을 taste.md의 "좋아하는 것" / "싫어하는 것" 섹션에 반영해라. 기존 내용은 지우지 말고 다듬거나 추가해라.
4. 무엇을 어떻게 바꿨는지 요약해서 알려줘. 표시된 평가가 3개 미만이면 "아직 데이터가 부족하다"고 말하고 업데이트하지 마라.
